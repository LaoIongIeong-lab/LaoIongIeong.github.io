Why analyze algorithms?

Constructing languages is perhaps the most powerful tool for abstraction. Learning about interpreters opens up new windows for further exploration in computer science and AI, such as:

- Search and logic programming. For my previous experience with these topics, see chapters 1-3, project "freshmanAdvisor".

- Why learn machine learning? "Machine learning solves the knowledge acquisition bottleneck problem, probability reasoning addresses the brittleness problem" from video 19:49.

- NetLogo is a dialect of Logo which itself can be modified from a Lisp interpreter.

- Purely functional Haskell vs. Lean (lazy evaluation vs. eager evaluation).

- ACT-R was implemented in Common Lisp.

- Douglas Hofstadter's Godel,Escher,Bach / I am a Strange Loop.
